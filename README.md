
## Starten der Anwendung

### Im Frontend-Ordner
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
### Im Backend-Ordner
```bash
npm start
```

- Tailwind CSS
- TypeScript
